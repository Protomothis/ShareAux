# Changelog

## v0.1.7

### 🎵 재생 품질 개선

- 곡 시작 시 처음부터 정상 재생 (이전에는 3초 뒤부터 재생되던 문제 수정) (#13)
- 재생 중 반복 끊김 현상 해결 — 네트워크 불안정 시 자동으로 버퍼를 다시 확보한 뒤 재생 (#13)
- 곡 전환 시 끝부분이 잘리지 않도록 서버가 충분히 대기 후 다음 곡으로 전환 (#13)
- 곡 전환 시 경과 시간이 순간적으로 이전 곡 시간으로 표시되던 문제 수정 (#8)
- 재생 시작까지 걸리는 시간 단축 (3초 → 2초) (#13)
- 버퍼링 중 재생 버튼에 로딩 표시 (#13)

### 🔍 검색 & 곡 정보

- 곡 정보(아티스트, 앨범) 매칭 정확도 대폭 향상 — 11% → 84% (#16)
- 검색 결과 무한스크롤이 동작하지 않던 문제 수정 (#20)
- 검색 재생목록 내 트랙에 즐겨찾기 버튼이 표시되지 않던 문제 수정 (#12)

### 🔒 보안

- WebSocket 연결 시 출처(Origin) 검증 추가 (#15)
- Swagger API 문서가 운영 환경에서 노출되지 않도록 차단 (#15)
- 비인증 상태의 대량 WebSocket 연결 시도 자동 차단 (#15)
- 인증 쿠키 구조 개선 — 민감 정보를 httpOnly JWT 기반으로 전환 (#21)
- 채팅 메시지에서 HTML 태그 문자 제거 (#15)

### 🛠 에러 처리 & 관리

- 로그인/회원가입 시 에러 메시지가 정확한 입력 필드에 표시 (#17)
- 어드민 인기 트랙 목록에 페이지네이션 추가 — 전체 트랙 관리 가능
- 어드민 트랙 목록에서 매칭된 곡명/아티스트명 우선 표시 (#16)

### 📱 모바일
