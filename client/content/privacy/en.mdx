## 1. Information We Collect

ShareAux collects the following information to provide its services.

### a. Google Login Users

- Google account identifier (unique ID)
- Email address
- Nickname (display name)
- Profile picture URL

This information is provided with user consent through Google OAuth 2.0 authentication. ShareAux does not access Google account passwords.

### b. Guest Users

- Nickname (entered by the user)

### c. Automatically Collected Information

- Room join/leave records
- Music play history (track title, play time)
- Chat messages (real-time in-room transmission, not permanently stored on server)

## 2. Purpose of Collection

- User authentication and session management
- Room participant identification and permission management
- Music play statistics and recommendation features
- Service operation and improvement

## 3. Retention Period

| Category | Retention Period |
|----------|-----------------|
| Google login users | Immediately deleted upon account deletion request |
| Guest users | Immediately deleted upon logout or session expiry |
| Chat messages | Kept in memory only, deleted when room closes (no permanent storage) |
| Play history | Deleted upon account deletion |

## 4. Third-Party Sharing

Collected personal information is not shared with third parties.

## 5. Google OAuth and External Services

ShareAux uses the following external services:

- **Google OAuth 2.0** — User authentication. Only email, name, and profile picture are requested; no other Google account data is accessed.
- **External media search API** — Music search. Does not access user external account data; only public search functionality is used.

When using Google services, the [Google Privacy Policy](https://policies.google.com/privacy) applies. Users can revoke ShareAux's access at any time from [Google Security Settings](https://security.google.com/settings/security/permissions).

## 6. User Rights

- You can request account deletion at any time to destroy all personal information.
- You can revoke ShareAux's OAuth access from Google Security Settings.
- You can request access to, correction of, or deletion of collected personal information.

## 7. Self-Hosting Notice

ShareAux is open-source self-hosting software. Personal data processing for each instance is the responsibility of the instance operator. This policy applies to the official instance.

## 8. Contact

For privacy-related inquiries, please contact the service administrator.
