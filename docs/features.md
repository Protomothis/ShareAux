# 기능 상세

## 방 시스템

### 방 생성

방을 만들 때 아래 항목을 설정할 수 있습니다.

- **방 이름**: 방 목록에 표시되는 이름
- **비공개 설정**: 비밀번호를 설정하면 입장 시 비밀번호 입력 필요
- **최대 인원**: 방에 동시 참여 가능한 인원 수
- **크로스페이드**: 곡 전환 시 페이드 효과
- **Auto DJ**: 큐가 비었을 때 자동으로 곡 추가
  - 모드: 인기곡 / 랜덤 / 유사곡
  - 트리거 임계값: 큐에 남은 곡 수 기준
- **기본 권한**: 새 참여자에게 큐 추가, 투표 스킵 허용 여부

### 방 참여

- 방 목록에서 클릭하여 입장
- 비공개 방은 비밀번호 입력 후 입장
- 초대 링크를 통한 직접 입장 지원
- 1인 1방 제한 — 다른 방에 입장하면 이전 방에서 자동 퇴장

### 호스트 (DJ)

방을 만든 사람이 호스트가 됩니다. 호스트는 다음 권한을 가집니다.

- 곡 건너뛰기 / 이전 곡
- 큐 순서 변경
- 참여자 권한 개별 설정
- DJ 권한 양도
- 참여자 강퇴
- 방 설정 변경

호스트가 퇴장하면 남은 참여자 중 한 명에게 자동 양도됩니다.

## 음악 재생

### 곡 검색

검색 모달에서 두 가지 방식으로 곡을 찾을 수 있습니다.

- **검색 탭**: 키워드로 검색. 자동완성 제안 지원
- **추천 탭**: 인기곡, 최근 재생곡, 좋아요 많은 곡 등 카테고리별 추천

한 번에 여러 곡을 선택하여 큐에 추가할 수 있습니다.

### 큐 관리

- 드래그 앤 드롭으로 순서 변경 (호스트 또는 권한 있는 유저)
- 곡 삭제 (본인이 추가한 곡 또는 호스트)
- 현재 재생 중인 곡 표시
- 다음 곡 프리로드 (곡 전환 지연 최소화)

### 투표 스킵

호스트가 아닌 참여자는 투표로 곡을 건너뛸 수 있습니다.

- 참여자 수의 과반수가 투표하면 스킵
- 곡 시작 후 일정 시간(5초) 이내에는 투표 불가
- 호스트는 즉시 스킵 가능

### Auto DJ

큐가 비었을 때 자동으로 곡을 추가하는 기능입니다.

- 방 설정에서 활성화/비활성화
- 모드: 인기곡 기반 / 랜덤 / 현재 곡과 유사한 곡
- 연속 실패 시 자동 비활성화 (시스템 메시지로 알림)

## 가사

### 싱크 가사

재생 중인 곡의 가사를 실시간으로 표시합니다.

- **줄 단위 싱크**: 현재 재생 위치에 맞춰 가사 줄 하이라이트
- **워드 단위 카라오케**: 단어별로 그라데이션 하이라이트 (Enhanced LRC 지원 곡)
- 스프링 물리 기반 스크롤 애니메이션
- 거리 기반 opacity/scale로 포커스 효과

### 번역

일본어, 영어, 중국어 가사를 한국어로 자동 번역합니다.

- Gemini AI 기반 번역
- 일본어 곡: 한글 발음 가이드 동시 제공
- 번역 완료 시 WebSocket으로 실시간 알림 → 자동 갱신
- 한국어 가사는 번역 생략

### 싱크 조절

가사 타이밍이 맞지 않을 때 수동으로 오프셋을 조절할 수 있습니다.

- ±100ms 단위 조절
- 곡별로 독립 적용

## 채팅 & 리액션

### 실시간 채팅

방 내 참여자 간 실시간 채팅을 지원합니다.

- WebSocket 기반 즉시 전송
- 최근 50개 메시지 히스토리 (재접속 시 복원)
- 도배 방지: 자동 감지 후 일시적 채팅 제한 (mute)
- 관리자/호스트는 mute 면제

### 리액션

플로팅 이모지 리액션으로 감정을 표현할 수 있습니다.

- 방 전체에 실시간 broadcast
- 애니메이션 효과로 화면에 떠오름

## 권한 시스템

2단계 권한 모델로 세밀한 접근 제어를 제공합니다.

### 계정 권한

초대 코드 생성 시 설정되며, 해당 코드로 가입/입장한 유저에게 영구 적용됩니다.

### 방 권한

호스트가 방 내에서 개별 유저에게 설정합니다. 멤버 목록에서 유저를 탭하면 권한 모달이 열립니다.

### 유효 권한

```
유효 권한 = 계정 권한 ∩ 방 권한
```

계정에서 제한된 권한은 방에서 열어줘도 사용할 수 없습니다.

### 권한 목록

| 권한 | 설명 | 기본 |
|------|------|------|
| 음악 듣기 | 오디오 스트리밍 수신 | 항상 활성 |
| 채팅 | 채팅 메시지 전송 | O |
| 리액션 | 이모지 리액션 전송 | O |
| 검색 | 곡 검색 | O |
| 큐 추가 | 큐에 곡 추가 | O |
| 순서 변경 | 큐 드래그 앤 드롭 | X |
| 투표 스킵 | 스킵 투표 참여 | O |
| 강퇴 | 다른 유저 강퇴 | X |
| 방 생성 | 새 방 만들기 | O |

## 게스트 시스템

### 초대 코드

관리자가 초대 코드를 생성하여 게스트를 초대합니다.

- 코드 자동 생성 또는 직접 입력
- 최대 사용 횟수 설정
- 만료일 설정 (선택)
- 코드별 계정 권한 설정
- 회원가입 허용 여부 설정

### 게스트 입장

- 초대 코드 + 닉네임만으로 입장
- 계정 생성 없이 즉시 참여
- 세션 만료 시 자동 삭제 (개인정보 미보관)

### 회원가입

초대 코드에 회원가입이 허용된 경우, 게스트가 정식 계정을 생성할 수 있습니다. 초대 코드에 설정된 권한이 계정에 적용됩니다.

## 관리자 백오피스

`/admin` 경로에서 접근할 수 있는 관리자 전용 페이지입니다.

### 대시보드

- 실시간 접속자 수 그래프 (시계열, 증분 업데이트)
- 서버 리소스 모니터링 (CPU, 메모리, ffmpeg 프로세스)
- 일별 재생 통계
- 유저 분포 (역할별, 가입 방식별)

### 유저 관리

- 유저 목록 (역할/가입 방식/상태 필터)
- 유저 상세: 프로필, 권한, 활동 이력
- 역할 변경, 계정 비활성화

### 방 관리

- 활성 방 목록 + 실시간 스트리밍 상태
- 방 상세: 설정, 멤버, 재생 이력

### 트랙 관리

- 트랙 랭킹 (재생 수, 좋아요, 점수)
- 트랙 상세: 메타데이터, 통계, 투표

### 초대 코드 관리

- 코드 목록, 사용 현황
- 코드 생성/삭제

### 시스템 설정

- 번역 설정 (활성화, 일일 제한, 모델)
- 기타 시스템 설정

### DB 정리

- 비활성 방, 오래된 재생 이력, 만료 초대 코드 등 정리
- 카테고리별 선택 정리

### 감사 로그

- 관리자 작업 이력 타임라인

### IP 차단

- IP 차단 목록 관리
- 차단된 IP는 미들웨어에서 즉시 거부

### 에러 로그

- 메모리 에러 로그 (최근 500건)
- 파일 에러 로그 (JSONL, 20MB 로테이션)

## 보안

- JWT 기반 인증 + 쿠키 refresh 토큰
- 비밀번호 bcrypt 해싱
- API 속도 제한 (ThrottlerGuard, 관리자 API 별도 제한)
- PoW(Proof of Work) CAPTCHA
- Helmet 보안 헤더
- 입력값 검증 (class-validator, whitelist 필터링)
- CORS 정책
- IP 차단 미들웨어
